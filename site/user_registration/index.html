  • Method: POST

  • Endpoint: https://api.example.org/api/v1/registration/create-registration/

  • User Authorization: Required 

      "Authorization": "Token xxxxxxxxxxxxxxxxxx"

  • Request: [JSON Body]

    {
  "first_name": "Kelvin",
  "middle_name": "Tetteh",
  "last_name": "Anyinatoe",

  "date_of_birth": "2017-02-10",
  "phone_number": 546424340,
}
