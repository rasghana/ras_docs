Pay for Ras account

  • Method: POST

  • Endpoint: https://api.example.org/api/v1/registration/pay-for-ras-account/

  • User Authorization: Required 

      "Authorization": "Token xxxxxxxxxxxxxxxxxxxxx"

  • Request: No [JSON Body]

    • Note: "name": "full name", "mobile": "+2332xxxxxx2" and "Order_id": "payment id" is gotten from Ras account registration details and Order_id is payment id.

  • Response

    • Note: token is already encoded and attached to redirect_url. All you need to do is to ship it
       {
   "payment": {
     "id": 63,
     "amount": "xxx",
     "is_ras_account": true,
     "event": null,
     "payment_status": "ip",
     "user": 1,
     "status_message": "",
     "transaction_no": "18273",
     "token": "xxxxxxxxxxxxxxxx",
     "created_date": "2017-04-08T18:18:48.855391Z",
     "modified": "2017-04-08T18:18:49.326173Z"
   },
   "redirect_url": "https://test.interpayafrica.com/interapi/confirmpayment?Token=xxxxxxxxxxxxxxxxxxxxxx"
 }

Pay for Event

  • Method: POST

  • Endpoint: https://api.example.org/api/v1/events/{event id}/make-event-payment/

  • User Authorization: Required 

      "Authorization": "Token xxxxxxxxxxxxxxxxxxxxx"

  • Request: No [JSON Body]

    • Note: "name": "full name", "mobile": "+2332xxxxxx2", "event":{event_id}, and "Order_id": "payment id" is gotten from Ras registration details. Order_id is payment_id and event_id is from Event.

  • Response

    • Note: token is already encoded and attached to redirect_url. All you need to do is to ship it. Once the paymet is completed, a call back url would sent from Interpay and the event would be automatically registered.
      {
  "payment": {
    "id": 59,
    "amount": "xxx",
    "is_ras_account": false,
    "event": 2,
    "payment_status": "ip",
    "user": 1,
    "status_message": "",
    "transaction_no": "18xx9",
    "token": "fxxxxxxxxxxxxxxxxxxxxxxx",
    "created_date": "2017-04-08T18:07:54.630529Z",
    "modified": "2017-04-08T18:07:55.109788Z"
  },
  "redirect_url": "https://test.interpayafrica.com/interapi/confirmpayment?Token=fxxxxxxxxxxxxxxxxxxxxx"
}
